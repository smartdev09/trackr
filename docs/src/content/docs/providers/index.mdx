Abacus supports multiple providers for tracking AI coding tool usage. Each provider is optional—configure only the ones you use.

<CardGrid>
  <Card title="Claude Code" icon="seti:config">
    Track token usage via the Anthropic Admin API. Includes cost tracking, model breakdown, and automatic API key mapping.

    [Set up Claude Code →](/trackr/providers/claude-code/)
  </Card>

  <Card title="Cursor" icon="seti:config">
    Import usage data via the Cursor Admin API or CSV export. Supports token counts, costs, and model breakdown.

    [Set up Cursor →](/trackr/providers/cursor/)
  </Card>

  <Card title="GitHub Commits" icon="github">
    Track AI-attributed commits across your organization. Detects Co-Authored-By headers from Claude, Cursor, Copilot, and more.

    [Set up GitHub Commits →](/trackr/providers/github-commits/)
  </Card>
</CardGrid>

## Provider Comparison

| Feature | Claude Code | Cursor | GitHub Commits |
|---------|-------------|--------|----------------|
| Token Usage | Yes | Yes | No |
| Cost Tracking | Yes | Yes | No |
| Model Breakdown | Yes | Yes | No |
| Commit Attribution | No | No | Yes |
| Real-time Sync | No (daily) | No (hourly) | Yes (webhook) |
| Historical Backfill | Yes | Yes | Yes (90 days) |

<Aside type="tip">
  For the most complete picture of AI adoption, we recommend configuring at least one usage provider (Claude Code or Cursor) alongside GitHub Commits.
</Aside>

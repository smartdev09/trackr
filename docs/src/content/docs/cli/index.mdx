Abacus includes a CLI for syncing data, managing identity mappings, and database operations.

## Usage

```bash
pnpm cli <command> [options]
```

## Quick Reference

| Command | Description |
|---------|-------------|
| `pnpm cli sync` | Sync usage from all providers |
| `pnpm cli github:sync <repo>` | Sync commits from a repository |
| `pnpm cli mappings` | List API key mappings |
| `pnpm cli stats` | Show database statistics |
| `pnpm cli db:migrate` | Run pending migrations |

## Getting Help

```bash
pnpm cli help
```
